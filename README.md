# 概要

本レポジトリは、拙著『パズルで鍛えるアルゴリズム力』の補足資料です。以下の内容を掲載しています。

- 本書で掲載しているパズルソルバー：solvers フォルダ以下
- 正誤表：errata.md

　

# 本書で使用する言語

本書では、パズルを解くアルゴリズムを実装するための言語として C++ を採用します。ただし基本的な機能のみを用いているため、ほかの多くの言語で同様の処理を実現できます。
